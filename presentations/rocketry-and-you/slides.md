## Contributing feature requests
- Create a task in the appropriate task list
- Create it in Stage: Backlog
- Will be reviewed, refined, and bumped to Ready
- Feel free to discuss in #rocketry-mission-control

---

## Improving Modules for Clients

Rocketry-First Development™
 
- Client has a need that is solved via a module update
- If possible, discuss in #rocketry-mission-control
- Branch the module in Rocketry
- Make your changes on that branch
- Add to client with the changes
- Open task in Backlog with link to PR and details of the solution

Note: This is a process that will be interactive and flexible - client may need updates but we may not want to mainline those changes to everyone.
